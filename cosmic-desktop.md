# COSMIC Desktop Keyboard Shortcuts

Reference for System76's COSMIC Desktop Environment (Rust-based DE for Pop!_OS and other Linux distributions).

## Launcher

| Shortcut | Action |
|----------|--------|
| Super | Open launcher |
| Super + / | Open launcher (alternate) |
| `filename` | Search recent files |
| `t:command` | Execute terminal command |
| `=equation` | Calculate equations |
| `?` | Display help menu |

## Window Navigation

| Shortcut | Action |
|----------|--------|
| Super + Tab | Switch between windows |
| Shift + Super + Tab | Switch windows in reverse |
| Super + Arrow Keys | Navigate between windows |
| Super + Alt + Arrow Keys | Navigate between displays |

## Window Management

| Shortcut | Action |
|----------|--------|
| Super + Q | Close window |
| Super + M | Maximize window |
| Super + F11 | Toggle fullscreen |
| Super + R | Resize window (use arrow keys) |
| Shift + Super + R | Resize window (alternate) |
| Super + S | Toggle window tabs |

## Tiling

| Shortcut | Action |
|----------|--------|
| Super + Y | Toggle auto-tiling |
| Super + G | Toggle floating mode for window |
| Super + O | Change window orientation |
| Super + X | Swap windows |
| Super + Enter | Move window into main position |

## Workspaces

| Shortcut | Action |
|----------|--------|
| Super + Ctrl + Arrow Keys | Navigate between workspaces |
| Super + 1-9 | Switch to workspace 1-9 |
| Super + 0 | Switch to last workspace |
| Shift + Super + Ctrl + Arrow Keys | Move window to adjacent workspace |
| Shift + Super + 1-9 | Move window to workspace 1-9 |

## Applications

| Shortcut | Action |
|----------|--------|
| Super + A | Open Applications Library |
| Super + W | Open Workspaces Overview |
| Super + F | Open Files |
| Super + T | Open Terminal |
| Super + B | Open default browser |

## Screenshots

| Shortcut | Action |
|----------|--------|
| Print | Screenshot (interactive) |
| Shift + Print | Screenshot active window |
| Ctrl + Print | Screenshot selected area |

## System

| Shortcut | Action |
|----------|--------|
| Super + Escape | Lock screen |
| Super + L | Lock screen (alternate) |
| Ctrl + Alt + Delete | Log out dialog |

## Accessibility

| Shortcut | Action |
|----------|--------|
| Super + Alt + S | Toggle screen reader |
| Super + = | Zoom in |
| Super + - | Zoom out |
| Super + . | Increase zoom |
| Super + , | Decrease zoom |

## Notes

- Super key is typically the Windows/Pop key
- Direction keys can be arrow keys or vim-style (h/j/k/l) in some contexts
- Shortcuts can be customized in Settings > Keyboard > Shortcuts
- COSMIC is still in active development; shortcuts may change between releases
